# Publicar um Aplicativo ASP.NET Core

Projeto desenvolvido durante os estudos de ASP.NET Core com o objetivo de aprender como publicar uma aplicação utilizando o .NET.

## Tecnologias

- C#
- ASP.NET Core
- .NET 10

## O que foi praticado

- Criação de uma aplicação ASP.NET Core
- Configuração para publicação da aplicação
- Geração dos arquivos de publicação
- Execução da aplicação publicada

## Como executar

Clone o repositório:

```bash
git clone https://github.com/Bressanvt/Publicar-um-aplicativo-ASP.NET-Core.git
```

Entre na pasta do projeto:

```bash
cd Publicar-um-aplicativo-ASP.NET-Core
```

Execute o projeto:

```bash
dotnet run
```

## Objetivo

Este projeto faz parte dos meus estudos em ASP.NET Core e foi desenvolvido para entender o processo de publicação de aplicações utilizando a plataforma .NET.
